Ready to submit your project? This guide walks you through everything you need for a complete submission.

## Submission Checklist

Before submitting, ensure you have:

- [ ] Public GitHub repository with MIT License
- [ ] Complete README with setup instructions
- [ ] 3 Vibe Session recordings (20-30 min each)
- [ ] Final demo video (3-5 min)
- [ ] Clean commit history showing development progress

## Step-by-Step Submission

<Steps>
1. **Finalize Your Code**
   
   Push all final changes to your main branch. Make sure the project runs without errors.

2. **Add MIT License**
   
   Create a `LICENSE` file in your repo root with the MIT License text.

3. **Complete Your README**
   
   Include:
   - Project title and description
   - Problem it solves
   - How to install and run
   - Technologies used
   - How AI was utilized

4. **Upload Vibe Sessions**
   
   Upload your 3 Vibe Session videos to YouTube (unlisted) or Google Drive.

5. **Create Final Demo**
   
   Record a polished 3-5 minute demo showcasing your project's features.

6. **Submit via Form**
   
   Complete the official submission form with all required links.
</Steps>

Use this structure for your README:

```markdown
# Project Name

Brief description of what your project does.

## Problem Statement

What problem does this solve?

## Demo

Link to your demo video.

## Features

- Feature 1
- Feature 2
- Feature 3

## Tech Stack

- Technology 1
- Technology 2

How did you use AI in development?

# Install dependencies

- Member 1 - Role

MIT License - see LICENSE file
```

## Video Requirements

    - Authentic, unedited footage
  </Card>
  <Card title="Final Demo" icon="star">
    - 3-5 minutes max

    - Explain the value proposition
  </Card>

<Aside type="tip" title="Video Hosting">
  Upload to YouTube (unlisted) or Google Drive. Make sure links are accessible to anyone with the link.
</Aside>

## Common Submission Mistakes

<Aside type="caution">
  Avoid these common issues that can disqualify your submission:
</Aside>

- **Private repository** — Must be public
- **Missing MIT License** — Required for prize eligibility
- **Incomplete videos** — All 3 Vibe Sessions required
- **Broken demo** — Test your project before submitting
- **Late submission** — Deadline is strictly enforced

## After Submission

1. You'll receive a confirmation email
2. Judges will review during the judging period
3. Winners announced on the final date
4. Prizes distributed to eligible winners
